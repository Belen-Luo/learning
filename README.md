## Development

**1. Install pnpm**

If you have not previously installed or used `pnpm`, you can install it by running the following command.

```bash
npm install pnpm -g
```

**2. Install Dependencies**

```bash
pnpm install
```

**3. Run the Project**

```bash
pnpm dev
```
