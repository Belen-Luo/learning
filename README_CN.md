## Development

**1. 安装 pnpm**

如果您以前没有安装或使用过' pnpm '，您可以通过运行以下命令安装它。

```bash
npm install pnpm -g
```

**2. 安装依赖关系**

```bash
pnpm install
```

**3. 启动项目**

```bash
pnpm dev
```
